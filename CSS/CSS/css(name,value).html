css(name,value)

在所有匹配的元素中，设置一个样式属性的值。
数字将自动转化为像素值
Set a single style property to a value on all matched elements.
If a number is provided, it is automatically converted into a pixel value.

返回值

jQuery

参数

name (value) : 属性名

value (String, Number) : 属性值

示例

将所有段落字体设为红色

jQuery 代码:

$("p").css("color","red");
