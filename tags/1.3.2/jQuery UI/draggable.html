draggable(options)

依赖性

  • ui.core.js

概述

让请求的对象可以拖动

这个jQuery插件可以让DOM节点随你鼠标拖动。你可以通过设置大量的选项，使其符合你的需求。

所有的回调函数(start,stop,drag)接受两个参数：浏览器原始的event对象和一个ui对象，你可以参考下面的文档来了解这个对象。(这里假定你把这个对象命名为"ui"):

  • ui.options - 用于初始化draggable的option参数
  • ui.helper - 指向用于被拖动的helper元素的jQuery对象
  • ui.position - helper元素当前相对于偏移元素的位置，是一个{ top, left }的对象
  • ui.absolutePosition - helper元素当前相对于当前页面的位置，是一个{ top, left }的对象

Creates new draggables on the nodeset supplied by the query.

This jQuery plugin makes DOM nodes draggable by mouse. You can additionally supply a great range of options to make them fit your needs.

All callbacks (start,stop,drag) receive two arguments: The original browser event and a prepared ui object, view below for a documentation of this object (if you name your second argument 'ui'):

  • ui.options - options used to initialize the draggable
  • ui.helper - the JQuery object representing the helper that's being dragged
  • ui.position - current position of the helper as { top, left } object, relative to the offset element
  • ui.absolutePosition - current absolute position of the helper as { top, left } object, relative to page

参数

appendToElement, Selector

For a draggable with a helper specified, the matched element passed to appendTo will be used as the helper's container. If not specified, the helper is appended to the same container as the draggable.

axisString

用于确定是水平方向(X)还是垂直方向(Y)拖动的，可选值：'x', 'y'。

cancelSelector默认值: ':input, button'.

如果你拖动的时候是点在匹配的元素上，则不会开始拖动。

containmentElement, Selector

cursor String

设置拖动时css的cursor值。

cursorAtObject, Array

用于设置拖动时光标的相对位置。传递一个Hash对象，可以设置top, left, right, bottom这几个属性。如{top:10,left:10}

delayInteger默认值: undefined

设置毫秒为单位的一个延时。用于想要点击一个元素而不想拖动这个元素时。

distanceInteger

以像素为单位，设置当鼠标按住移动多少距离后才开始拖动

grid[Integer x, Integer y]

使拖动的距离栅格化。设定x和y来决定每次能拖动的距离。

handleElement, Selector默认值:调用draggable的元素

允许决定鼠标在哪个元素上才可以开始拖动。类似于一个把手。

helperString, Function默认值：'original'

允许一个helper元素用于拖动时的显示。设置为"clone"会使拖动时复制一份元素出来。可选值：'original', 'clone',函数。如果提供了一个函数，那必须返回一个标准合法的DOM节点。

opacityFloat

设置helper元素拖动时候的透明度。

revertBoolean,String

如果设为true，那么元素在拖动结束后会返回他开始拖动时的位置。这个参数也可以设置为"valid"或者"invalid"。如果设置为"invalid"，那么仅仅在拖动的元素没有被放到可置放的对象上时候返回原位。如果设为"valid"，那就相反。

revertDurationInteger默认值:'500'

返回原位的动画持续时间,单位毫秒。

scrollBoolean默认值:true

如果设为真，则在拖动的时候，容器自动滚动。

scrollSensitivityInteger默认值:20

滚动敏感距离。设置当在离视口边缘多少像素内时开始滚动。这个距离是相对于鼠标的而不是拖动元素的。

scrollSpeedInteger默认值:20

设置当鼠标移动到滚动敏感距离内的时候，容器滚动的速度。

snapBoolean,Selector

snapMode"inner","outer"

snapTolerance Integer

refreshPositions Boolean

如果设置为true，每移动一下鼠标，所有拖动的位置将重新计算。注意：这用于高度动态的页面，但极大降低性能。

zIndex Integer

设置被拖动的helper对象的z-index值。

start Function(e, ui)

当拖动开始时调用的函数。

drag Function(e, ui)

当拖动拖动中调用的函数。

stop Function(e, ui)

当拖动停止时调用的函数。

示例

<script>
    $(document).ready(function(){
        $(".block").draggable();
    });
</script>
<style>
    .block { 
        border: 2px solid #0090df;
        background-color: #68bfef;
        width: 150px; 
        height: 70px;
        margin: 10px; 
    }
</style>
<div class="block"></div>

draggable("disable")

依赖性

  • ui.core.js

概述

临时性禁用拖动功能

这个方法临时性地禁用拖动功能。之后可以用$(..).draggable("enable")重新启用。

Temporarily disable the draggable functionality.

This method temporarily disables draggable functionality. It can later be reenabled by calling $(..).draggable("enable").

参数

"disable"String

draggable("enable")

依赖性

  • ui.core.js

概述

启用拖动功能。

这个功能可以把先前禁用的拖动功能重新启用

Enable the draggable functionality.

This method re-enables a previously disabled draggable.

参数

"enable"String

draggable("destroy")

依赖性

  • ui.core.js

概述

彻底移除对象上的拖动功能

这个方法将会把对象上的拖动功能彻底移除。

Completely remove the draggable functionality.

This method removes the draggable functionality completely.

参数

"destroy"String
