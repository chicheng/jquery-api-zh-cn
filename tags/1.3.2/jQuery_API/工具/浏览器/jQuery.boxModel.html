jQuery.boxModel

当前页面中浏览器是否使用标准盒模型渲染页面。 W3C CSS 盒模型.
States if the current page, in the user's browser, is being rendered using the W3C CSS Box Model.

返回值

Boolean

示例

在 Internet Explorer 怪癖模式（QuirksMode）中返回 False。

jQuery 代码:

$.boxModel
