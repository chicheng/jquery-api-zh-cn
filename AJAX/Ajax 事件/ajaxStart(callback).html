ajaxStart(callback)

AJAX 请求开始时执行函数。Ajax 事件。
Attach a function to be executed whenever an AJAX request begins and there is none already active. This is an Ajax Event.

返回值

jQuery

参数

callback (Function) : 待执行函数

示例

AJAX 请求开始时显示信息。

jQuery 代码:

 $("#loading").ajaxStart(function(){
   $(this).show();
 });
