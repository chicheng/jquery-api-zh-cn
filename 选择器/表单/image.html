:image

匹配所有图像域
Matches all input elements of type image.

返回值

Array<Element>

示例

匹配所有图像域

HTML 代码:

<form>
  <input type="text" />
  <input type="checkbox" />
  <input type="radio" />
  <input type="image" />
  <input type="file" />
  <input type="submit" />
  <input type="reset" />
  <input type="password" />
  <input type="button" />
  <select><option/></select>
  <textarea></textarea>
  <button></button>
</form>

jQuery 代码:

$(":image")

结果:

[ <input type="image" /> ]
