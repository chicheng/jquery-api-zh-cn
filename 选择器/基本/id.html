#id

根据给定的ID匹配一个元素。
Matches a single element with the given id attribute.

返回值

Element

参数

id (String) : 用于搜索的，通过元素的 id 属性中给定的值

示例

查找 ID 为"myDiv"的元素。

HTML 代码:

<div id="notMe"><p>id="notMe"</p></div>
<div id="myDiv">id="myDiv"</div>

jQuery 代码:

$("#myDiv");

结果:

[ <div id="myDiv">id="myDiv"</div> ]
